# Field Examples — Esports / Gender Performance Gap Literature

**Artifact type:** Worked example / demonstration artifact
**Purpose:** Demonstrate prompt output and audit workflow on a real contested claim
**Author role:** Domain framing, source selection, prompt routing, adversarial review, final judgment
**AI role:** Structured audit generation, synthesis, counterargument generation, formatting

-----

This folder contains the full audit trail for the Esports / Gender Performance Gap Literature worked example. It intentionally preserves the structured, audit-like format produced by the prompts so readers can see how the workflow operates. The goal is transparency and replicability.

These are not conventional essays. They are demonstration artifacts showing what happens when the adversarial research prompts are run against a real contested claim in a real field.

## How to replicate

1. Take the foundational paper or claim identified in the worked example
1. Run the prompts in the recommended sequence (see README.md in root)
1. Bring your own domain knowledge conversationally between steps
1. The prompts structure the audit. The domain knowledge is what makes it hard to dismiss.
